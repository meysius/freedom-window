# Freedom Window

A simple HTML viewer for Telegram channel crawl data. No installation or internet connection required — just open the file in your browser.

---

## How to Use (English)

### Step 1 — Download the project

1. Go to the GitHub repository page in your browser.
2. Click the green **Code** button near the top right.
3. Select **Download ZIP**.
4. Once downloaded, find the ZIP file (usually in your **Downloads** folder).
5. Right-click the ZIP file and choose **Extract All** (Windows) or double-click it (Mac) to unzip it.

### Step 2 — Open the file

1. Open the extracted folder.
2. Find the file named **`window.html`**.
3. Double-click it — it will open in your default web browser.
4. That's it! No setup, no installation needed.

> **Tip:** Works in Chrome, Firefox, Edge, and Safari.

## نحوه استفاده (فارسی)

### مرحله اول — دانلود پروژه

1. به صفحه مخزن گیت‌هاب بروید.
2. روی دکمه سبز رنگ **Code** در بالای صفحه کلیک کنید.
3. گزینه **Download ZIP** را انتخاب کنید.
4. فایل ZIP دانلود شده را پیدا کنید (معمولاً در پوشه **Downloads** است).
5. روی فایل ZIP راست‌کلیک کرده و **Extract All** را بزنید (ویندوز)، یا در مک روی آن دابل‌کلیک کنید تا از حالت فشرده خارج شود.

### مرحله دوم — باز کردن فایل

1. پوشه استخراج‌شده را باز کنید.
2. فایلی با نام **`window.html`** را پیدا کنید.
3. روی آن دابل‌کلیک کنید — فایل در مرورگر پیش‌فرض شما باز می‌شود.
4. همین! نیازی به نصب یا اینترنت نیست.

> **نکته:** در مرورگرهای Chrome، Firefox، Edge و Safari کار می‌کند.
